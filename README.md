Whatsapp, Telegram, Signal, etc.

<!-- GETTING STARTED -->
## Getting Started

You can run the project with docker or npm.



### Installation

#### Docker
<br>
If you want to run it via docker go to packackes on the right side of the Github page and follow the instructions.
<br></br>

#### NPM
1. Clone the repo
   ```sh
   git clone https://github.com/solun-pm/solun-webmail.git
   ```

2. Install NPM packages
   ```sh
   npm i
   ```
3. Build the App
   ```sh
   npm run build
   ```
4. Start the App
   ```sh
   npm run start
   ```
